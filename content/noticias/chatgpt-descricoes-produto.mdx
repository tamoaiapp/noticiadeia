Escrever descrições de produto é uma das tarefas mais repetitivas do e-commerce. Com o ChatGPT, você cria descrições otimizadas para SEO e para conversão em segundos. Veja como.

## Por que a descrição de produto importa tanto

Uma boa descrição de produto:
- Aparece nas buscas do Mercado Livre e Google
- Responde as dúvidas do comprador antes que ele precise perguntar
- Aumenta a taxa de conversão (mais vendas com o mesmo tráfego)
- Reduz devoluções (quando o cliente sabe exatamente o que está comprando)

Muitos vendedores copiam a descrição do fornecedor — que é genérica, mal escrita e igual a dezenas de concorrentes. Com o ChatGPT, você cria algo único em segundos.

## Prompts prontos para usar

### Para Mercado Livre (foco em SEO)

```
Crie uma descrição de produto para o Mercado Livre.
Produto: [nome do produto]
Principais características: [liste 3-5 características]
Público-alvo: [quem compra]
Tom: profissional mas acessível
Inclua: 
- Título com as palavras-chave principais
- Bullet points com os benefícios (não apenas características)
- Sessão "Para quem é este produto"
- Informações técnicas no final
Limite: 400 palavras
```

### Para Shopee (foco em conversão)

```
Escreva uma descrição de produto para a Shopee.
Produto: [nome]
Características: [liste]
Use emojis relevantes para destacar pontos chave.
Formato: texto curto e objetivo, fácil de escanear.
Inclua: 
- 3 principais benefícios em destaque
- Especificações técnicas em lista
- Call-to-action no final
Tom: jovem e direto
```

### Para loja própria/site (foco em SEO Google)

```
Crie uma descrição de produto para SEO.
Produto: [nome]
Categoria: [categoria]
Palavra-chave principal: [keyword]
Escreva:
1. Meta description (150 chars)
2. Título H1 com keyword
3. Descrição longa (300 palavras) com keyword integrada naturalmente
4. 5 bullet points de benefícios
Tom: persuasivo mas honesto
```

## Exemplo prático

**Produto**: Fone de ouvido Bluetooth sem fio

**Prompt**:
```
Crie uma descrição de produto para Mercado Livre.
Produto: Fone de ouvido Bluetooth TWS
Características: até 6h de bateria, cancelamento de ruído passivo, 
resistente à suor, conexão Bluetooth 5.0, microfone integrado.
Público-alvo: pessoas que praticam exercícios e precisam de fone prático.
```

**Resultado gerado pelo ChatGPT**:
> **Fone Bluetooth TWS — Som Perfeito para seus Treinos**
>
> Cansado de fone caindo durante o exercício? O Fone TWS foi feito para quem não para. Com Bluetooth 5.0 de última geração, a conexão é instantânea e estável — sem travamentos mesmo nas corridas mais intensas.
>
> ✅ **6 horas de música ininterrupta** — vai e volta no treino sem recarregar
> ✅ **Resistente ao suor** — pode usar na academia, corrida ou ciclismo
> ✅ **Microfone integrado** — atenda chamadas sem tirar o fone do ouvido
> ✅ **Bluetooth 5.0** — conexão estável em até 10 metros de distância

## Como refinar o resultado

Se o resultado não ficou como você queria, use continuações:

- "Reescreva com tom mais informal"
- "Adicione mais ênfase no benefício de bateria"
- "Reduza para 150 palavras mantendo os pontos principais"
- "Adapte para o público feminino"

O ChatGPT lembra o contexto da conversa — você pode iterar até chegar no texto ideal.

## Dica: crie um modelo padrão da sua loja

Depois de criar descrições para vários produtos, você vai notar um padrão que funciona bem para o seu nicho. Crie um "mega prompt" com o estilo da sua loja e use como base para todos os produtos novos.
