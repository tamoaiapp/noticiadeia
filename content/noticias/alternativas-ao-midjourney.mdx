## FAQ — Midjourney

**O Midjourney é gratuito?**
Sim, existe uma versão gratuita: sem plano gratuito. Para uso profissional intenso, o plano pago (US$10/mês (Basic)) traz limites maiores e acesso ao modelo mais avançado.

**O Midjourney funciona em português?**
Sim. A ferramenta tem bom desempenho em português brasileiro. Para melhores resultados, escreva seus prompts em PT-BR e especifique quando precisar de linguagem localizada para o Brasil.

**Preciso de cartão de crédito para criar conta?**
Não é necessário cartão de crédito para o plano gratuito. O cadastro é feito com e-mail em menos de 2 minutos.

**O Midjourney é seguro para dados da minha empresa?**
A Midjourney Inc. tem políticas de privacidade claras, mas evite inserir dados sensíveis de clientes (CPF, dados bancários, informações médicas) sem revisar os termos de uso. Para uso empresarial, verifique se existe versão enterprise com garantias adicionais de privacidade.

**Qual é a diferença entre o Midjourney gratuito e o pago?**
A versão gratuita tem limites de uso e acessa modelos menos avançados. O plano pago (US$10/mês (Basic)) oferece velocidade superior, janela de contexto maior, acesso prioritário e funcionalidades extras como geração de imagens ou análise de arquivos.

**O Midjourney tem aplicativo para celular?**
Sim, o Midjourney tem apps para iOS e Android disponíveis nas lojas oficiais. A experiência é bastante comparável à versão web para uso diário.

**Como o Midjourney se compara às alternativas?**
O Midjourney se destaca por qualidade fotográfica e artística de imagens. Alternativas como ChatGPT, Claude e Gemini têm pontos fortes diferentes — a melhor escolha depende do seu caso de uso específico.
